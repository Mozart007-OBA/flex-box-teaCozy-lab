## Tea Cozy - Site Web d’un Salon de Thé

**Tea Cozy** réalisé dans le cadre de ma formation en développement web.

### Description

Ce projet consiste en la création d’un site web complet pour un salon de thé fictif nommé **"Tea Cozy"**.

J’ai reproduit fidèlement le design fourni (design spec) en utilisant uniquement **HTML5** et **CSS3**, avec une mise en page basée sur **Flexbox**.
